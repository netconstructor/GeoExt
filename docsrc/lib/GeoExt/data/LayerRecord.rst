:class:`GeoExt.data.LayerRecord`
================================

.. class:: LayerRecord

    Create a record representing an ``OpenLayers.Feature.Vector`` object.
    This layer will always have at least the following fields:
    
    * layer ``OpenLayers.Layer``
    * title ``String``
